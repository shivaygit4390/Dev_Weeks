# Revision Question Bank Guide

## Purpose

This folder is not for first-time learning.
It is for recall testing.

Main file:

1. `QuestionBank.md`

---

## Best Way To Use It

1. finish one theory section
2. open the question bank
3. answer aloud without reading solution material
4. note weak zones
5. go back only to that section

---

## Use Cases

- interview revision
- end-of-day self check
- weekend recall test

---

## Done When

- you can answer a good chunk without freezing
- you can identify weak topics quickly instead of re-reading everything
