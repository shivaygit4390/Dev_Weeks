## Weekly Success Definition

Week 1 is in a good state if you can do these without looking too much:

- explain stack vs heap
- explain closure using memory and scope
- explain shallow vs deep copy
- explain promise flow and async/await flow
- explain event loop order
- write simple debounce and throttle
- explain default vs named exports
- answer rapid-fire questions from the question bank

You do not need perfect wording.
You do need repeatable clarity.

---

## 3 Pass Revision Loop

### Pass 1 - Understand

- read note
- mark confusing line
- say concept aloud

### Pass 2 - Trace

- open practice file
- predict output or flow
- trace variable and function movement

### Pass 3 - Rebuild

- close file
- write from memory in `test.js`
- compare with original

Repeat the loop instead of collecting more theory.
